# Ejercicios de Python y Lógica de Programación

Este repositorio es una colección de scripts y ejercicios de Python desarrollados de forma autodidacta. 
El objetivo principal de este proyecto es dominar los fundamentos de la sintaxis del lenguaje, 
el diseño de algoritmos y la resolución de problemas lógicos.

## 🎯 Objetivos del Proyecto
- Dominar la sintaxis principal de Python, sus funciones integradas y las mejores prácticas de escritura de código.
- Desarrollar un pensamiento lógico-matemático y habilidades sólidas para resolver problemas de programación.
- Practicar la manipulación de datos, el control de flujo y la optimización de código utilizando Python.

## 📋 Conceptos Clave Cubiertos
A través de estas prácticas, he implementado y reforzado los siguientes fundamentos de las ciencias de la computación:
- **Control de Flujo y Lógica:** Uso de condicionales complejos (`if/elif/else`) y bucles (`for`, `while`) para controlar la ejecución del programa.
- **Estructuras de Datos:** Manipulación de colecciones nativas de Python como **Listas**, **Tuplas**, **Diccionarios** y **Conjuntos** (Sets) para gestionar información de manera eficiente.
- **Funciones y Código Modular:** Escritura de funciones reutilizables, limpias y estructuradas con un manejo correcto de parámetros.
- **Manipulación de Cadenas (Strings):** Técnicas de procesamiento, análisis y formateo de texto.
- **Lógica Matemática:** Resolución de problemas clásicos de algoritmos que involucran operaciones aritméticas, series y filtrado de datos.

## 📝 Documentación del Código
Todos los scripts dentro de este repositorio incluyen comentarios detallados en español. En ellos se explica paso a paso el proceso de pensamiento, la estructura lógica y la mecánica detrás de cada solución para garantizar la legibilidad y el orden del código.

## 🛠️ Tecnologías Utilizadas
- **Python 3.x**
- **Entorno de desarrollo:** Visual Studio Code / Ejecución en Terminal
